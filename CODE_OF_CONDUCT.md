# Code of Conduct

This project adopts the [Contributor Covenant, version 2.1][covenant]. The full
text and rationale are maintained at the canonical URL below; please read it
before participating.

[covenant]: https://www.contributor-covenant.org/version/2/1/code_of_conduct/

## In short

- Be respectful in issues, pull requests, and review comments.
- Critique ideas, not people.
- Assume good faith from contributors; ask before escalating.
- The maintainers reserve the right to remove comments, close issues, and ban
  participants whose behavior violates the linked Covenant.

## Reporting

If you experience or witness unacceptable behavior, report it privately via the
channel in [SECURITY.md](SECURITY.md) and label it as a conduct report. The
maintainers will review confidentially. Do **not** post the report as a public
issue.

## Scope

This applies to project spaces (issues, PRs, discussions, commit messages) and
to public spaces when an individual is representing the project.
